# Apfel Companion

Apfel Companion is a native macOS chat client for running LLMs locally via [apfel](https://github.com/Arthur-Ficial/apfel?tab=readme-ov-file).

## Features

- Chat interface with real-time streaming responses
- Automatic apfel server management (starts on launch, stops on quit)
- Connects to an existing apfel instance if one is already running

## Prerequisites

- macOS 14+
- [apfel](https://github.com/alexscheid/apfel): `brew install apfel`
